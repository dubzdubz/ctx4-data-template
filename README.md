# ctx4.ai Context Data Template

This is a template repository for storing your personal context data used by [ctx4.ai](https://ctx4-ai.vercel.app/).

## Getting Started

1. Click "Use this template" to create your own context repository. **Make it private!**.
2. Sign up at [ctx4.ai](https://ctx4-ai.vercel.app/) and connect this repository
3. (Optional) Customize the files in this repository to add your personal context, preferences, and knowledge
4. Your AI assistant will use this context to provide more personalized and relevant help

## Repository Structure
This is an example and starting point. Ask your agent customize it for you!

- **`resources/`** - Core instructions and context for your AI assistant
  - `instructions.md` - How you want your AI to behave and work with you
  - `context.md` - Background information about you and your work
- **`skills/`** - Custom skills that extend your AI's capabilities
  - `skill-creator/` - Built-in skill for creating new skills
- **`knowledge/`** - Personal knowledge base and memories
  - `personal.md` - Information about you, your preferences, and work style
  - `agent/learnings.md` - Documented learnings for the agent

## Learn More

Visit [ctx4.ai](https://ctx4-ai.vercel.app/) to connect your repository and start using it with your AI assistant.
